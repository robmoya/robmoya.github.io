You can contribute to Scientific Progress

Everything just by generating Boid Power

Scientist need to sort through massive amounts of data in order to solve big important problems

Large tasks are broken up into small work units that can run on any computer or android phone

Work units are distributed to users computers all over the world

Computers crunch work units, and send the results back to the researchers

Finished work units are assembled back together to form a final answer

Users running Boid generate Boid power when their devices return valid work units
